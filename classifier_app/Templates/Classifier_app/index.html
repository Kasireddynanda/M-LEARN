Supervised learning is a machine learning approach where models are trained using labeled data, meaning the input data is paired with corresponding correct outputs. The algorithm learns by mapping inputs to outputs and adjusts itself based on errors.
Classification

Used when the output is categorical. The model learns to assign input data to predefined categories.
Regression

Used when the output is continuous. The model learns to predict a numerical value based on input data.
→

Unsupervised learning is a type of machine learning where the model learns patterns from unlabeled data. It is mainly used for discovering hidden structures in data. Instead of learning from predefined outputs, it groups similar data points through clustering or simplifies data using dimensionality reduction
Clustering

Groups similar data points into clusters based on their characteristics.
Dimensional Reduction

Reduces the number of features while retaining important information.
→

Reinforcement Learning (RL) is a type of machine learning where an agent learns to make decisions by interacting with an environment. Instead of learning from labeled data, the agent learns by trial and error, receiving rewards or penalties based on its actions. The goal is to maximize cumulative rewards over time.
